# Documentation Website

This is the documentation website for React Native Reanimated DnD, built with [Mintlify](https://mintlify.com).

### Installation

```
npm install
```

### Local Development

```
npx mintlify dev
```

This starts a local development server at `http://localhost:3000`. Changes are reflected live without restarting.

### Deployment

Push to your connected GitHub repository. Mintlify automatically builds and deploys on push.
